HTML Authentication POC

Description
This project is a basic HTML Authentication Proof of Concept (POC) created using plain HTML only.
It demonstrates page navigation between authentication-related pages using anchor tags.

No CSS or JavaScript has been used in this project as per the assignment instructions.

Pages Included
login.html
register.html
forgot-password.html
reset-password.html
dashboard.html

Navigation Flow
Login → Dashboard
Login → Register
Login → Forgot Password
Forgot Password → Reset Password
Register → Login
Reset Password → Login
Dashboard → Login (Logout)

All redirections are implemented using anchor (<a>) tags.

Technologies Used
HTML5 (Only)
